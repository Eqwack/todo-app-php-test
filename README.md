# 📝 ToDo App en PHP

Une petite application de gestion de tâches réalisée en PHP, HTML/CSS, et MySQL.

## 🚀 Fonctionnalités

- Ajouter, modifier, supprimer des tâches
- Rechercher une tâche par titre ou description
- Stockage en base de données MySQL
